# 動詞活用

1. 一类。
   1. 不以[**る**]结尾
   2. 特殊的一类动词（符合二类动词特征，实质是一类动词）[帰る、入る、ある、切る、要る...]
2. 二类。
   1. 以[**る**]结尾
   2. 倒数第二个假名在[**え**]和[**い**]上
3. 三类。
   1. くる
   2. する

## 一 [て]形

> 14 - 16课

原型变法

1. 一类动词

| 最后一个音节 | 变法   | 举例         |
| ------------ | ------ | ------------ |
| う、つ、る   | 促音变 | [って]       |
| む、ぶ、ぬ   | 拨音变 | [んで]       |
| く、ぐ       | い音变 | [いて、いで] |
| す           | して   | [して]       |

2. 二类动词
   - 去[る]加[て]
     > 食べて、みて、見せて
3. 三类动词
   1. 来ます　 → きて
   2. します　 → して

## 二 [ない]形

> 17课

1. 一类动词
   1. 最后一个变[あ]段加[ない]
   2. 特殊：以[う]变[わ]
2. 二类动词
   1. 去[る]加[ない]
3. 三类动词
   1. する　→ しない
   2. くる　→ **こない**　（特殊）
